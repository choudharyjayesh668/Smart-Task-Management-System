# Smart Task Management System


A Java console-based application to manage tasks with priority and status.


## Features
- Add new tasks
- View all tasks
- Update task status
- Filter high-priority tasks


## Technologies Used
- Java
- OOP Concepts
- Collections (ArrayList)
- VS Code

## How to Run
1. Compile the program:
   javac TaskManager.java
2. Run the program:
   java TaskManager
